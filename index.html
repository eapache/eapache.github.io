    Development

    I am a developer at Shopify, and a core developer of Wireshark. I designed and wrote the original version of the Sarama library. I am responsible for a couple of tiny pieces of Linux kernel code. My other projects include:

    • Starscope — A code navigation tool similar to cscope or ctags, but for Ruby, Golang and other languages. It can trace function calls, variable references, and more, as well as export to cscope's database format for integration with editors like Vim.
    • Channels — A collection of helper functions and special types for working with and extending Golang's existing channel type.
    • Go-Resiliency — A collection of resiliency patterns implemented for Golang, based partially on Netflix's Hystrix library.
    • Vim-YAFF — Yet Another Fuzzy Finder, a plugin for Vim. Similar to Ctrl-P, Unite, etc. but simpler and faster.
    • Vim-Rainbow — Rainbow parentheses for Vim. Kien's original version was no longer being maintained, so I forked this version and carried on their good work.
    • Wheel-of-Fortune Allocator — A blazing-fast C/C++ memory allocator with a near-constant-time "free-all" operation. Currently in use by Wireshark.

    Most of the above list I consider "done", but that doesn't mean unmaintained. I am happy to accept pull requests or bug reports for any of my projects (including the ones listed on GitHub but not big/interesting enough to be listed here).
